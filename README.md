테슬라 Model Y 할부 시뮬레이터
tesla_cal은 Streamlit 기반 웹 앱으로, 테슬라 Model Y 구매 시 예상 비용(출고가 + 옵션 + 보조금 + 할부 + 유지비)을 빠르게 계산해주는 계산기입니다.

🔍 주요 기능
모델 트림 선택
Model Y Premium RWD
Model Y Premium Long Range AWD
옵션 선택
도색, 휠, 인테리어 추가 옵션 반영
보조금 계산
국가 보조금 + 지역(서울, 경기, 부산 등) 보조금
슬라이더로 보조금 직접 조정 가능
할부 시뮬레이션
선수금 비율, 기간(24/36/48/60), 연이율 조절
월 납입금, 이자, 원금 계산
월 유지비
보험료, 충전비, 기타 유지를 월단위 입력
결과
실구매가
월 총비용
총 이자
기간별 비교 테이블 (24~60개월)
5년 총 보유 비용
⚙️ 요구 사항
Python 3.8+
streamlit
📦 설치
requirments.txt 이름이 requirements.txt가 아닌 점 확인해 주세요. 프로젝트 파일명 그대로 사용해야 합니다.

▶️ 실행
브라우저에 자동 오픈되며, 로컬 기본 주소는 http://localhost:8501입니다.

🧩 사용법
트림 선택에서 모델/출고가 설정
옵션에서 원하는 도색·휠·인테리어 선택
보조금 슬라이더로 국가/지방 보조금 조정
선수금, 할부 기간, 연이율 입력
보험/충전/기타 월 유지비 입력
우측에서 계산 결과 확인
실구매가
월 납입 상세
기간별 총비용 비교
